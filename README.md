# Desafio TEIA 2024

## Introduction

Use `Node.js` and `Express` to build an API that count characters of a string and define if it is a palindrome word.

## Instructions

### Project Setup 

Please perform npm install

### Start the API

Perform npm start

| - | ------ | ----------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
**post**                                                                               |
| 1 | POST   | /api/manipulacao-string              | perform operations over string
